# Web Development Portfolio

This repository contains two distinct web development projects showcasing different technologies and approaches.

## 📁 Project Structure

### 📂 project2-giphy-shed/

**GIF Search Application** - HTML, CSS, JavaScript

- Responsive web app using Giphy API
- Vanilla JavaScript implementation
- Mobile-first design approach
- [View Project 2 README](./project2-giphy-shed/README.md)

### 📂 project3-react-todo-app/

**React Todo Application** - React, React Router, Modern CSS

- Full-featured todo management system
- React functional components with hooks
- Contact form with validation
- Responsive design with accessibility features
- [View Project 3 README](./project3-react-todo-app/README.md)

## 🚀 Quick Start

### Project 2 (Giphy Shed)

```bash
cd project2-giphy-shed
# Open index.html in your browser
```

### Project 3 (React Todo App)

```bash
cd project3-react-todo-app
npm install
npm start
```
