# Project 2: Giphy Shed 🎬

A responsive web application for searching and viewing GIFs using the Giphy API. Built with HTML, CSS, and vanilla JavaScript.

## 🌟 Features

- **GIF Search**: Search for GIFs using keywords
- **Responsive Gallery**: Grid-based display that adapts to screen size
- **Mobile Navigation**: Hamburger menu for mobile devices
- **API Integration**: Real-time GIF fetching from Giphy API
- **Error Handling**: Graceful handling of failed requests
- **Loading States**: User feedback during API calls

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with Flexbox and Grid
- **JavaScript ES6+** - API integration and interactivity
- **Giphy API** - GIF search and retrieval

## 📦 Installation Instructions

### Prerequisites

- Modern web browser
- Internet connection (for API calls)

### Setup

1. **Clone or download the repository**
2. **Navigate to the project2-giphy-shed folder**
3. **Open `index.html` in your web browser**
4. **Start searching for GIFs!**

### API Setup

The Giphy API key is already included for demonstration purposes. For production use:

1. Get your own API key from [Giphy Developers](https://developers.giphy.com/)
2. Replace the API_KEY in `app.js`
